### Budget Management
1. Click "Budgets" tab
2. Add budget with category & limit
3. System tracks spending
4. Get alerts at thresholds
5. View progress in real-time

### Financial Reports
1. Click "Reports" tab
2. Select date range
3. View summary + breakdown
4. Export as CSV/JSON
