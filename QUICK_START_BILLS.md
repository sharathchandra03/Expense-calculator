# Quick Start: Bills Manager 🧾

## 60-Second Setup

1. **Open FinanceOS**
2. **Tap Bills tab** (📅 at bottom)
3. **Tap "+ Add New Bill"**
4. **Fill in**:
   - Title: "Netflix"
   - Amount: 15.99
   - Due Date: 15th
   - Category: Subscription
5. **Tap ✓ Save Bill** ✅

That's it! Bill is now tracked.

---

## What You Can Do Now

### Add Bills ✅
```
+ Add New Bill → Fill form → Save
```

### Edit Bills ✅
```
Find bill → ✏️ Edit → Change → Save
```

### Delete Bills ✅
```
Find bill → 🗑️ Trash → Confirm → Done
```

### Mark Paid ✅
```
Find bill → ✓ Check → Turns green
```

### Search Bills ✅
```
Type in search box → See matching bills
```

### Filter Bills ✅
```
Tap "All" / "Unpaid" / "Paid" → See filtered list
```

## Common Tasks

### Add Monthly Subscription
1. Tap "+ Add New Bill"
2. Title: [Service name]
3. Amount: [Monthly cost]
4. Due Date: [Day of month]
5. Category: Subscription
6. ✓ Check "Recurring"
7. Save

### Pay a Bill
1. Find bill in list
2. Tap ✓ check circle
3. Watch it turn green
4. Total due decreases 📉

### Track Annual Bill
1. Add bill normally
2. Amount: yearly cost
3. Category: appropriate type
4. Due date: when it's due
5. Leave "Recurring" unchecked
6. Save

## Troubleshooting

**Can't find Add button?**
→ Look for "+ Add New Bill" at bottom of screen

**Won't save bill?**
→ Check: Title and Amount filled? Amount > 0?

**Can't find ✓ button?**
→ Hover over bill or swipe left on mobile

**Want to undo delete?**
→ Can't undo - use caution! But you can add again

**Amounts wrong?**
→ Verify all bills marked paid/unpaid correctly
