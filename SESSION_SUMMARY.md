## 💡 Design Decisions

### Architecture Choices
1. **Service Layer**: Business logic separated from UI
2. **Repository Pattern**: Clean data access
3. **Component Composition**: Reusable, maintainable components
4. **Framer Motion**: Smooth, performant animations
5. **IndexedDB**: Fast, local data storage

### UI Choices
1. **Bottom Navigation**: Thumb-friendly on mobile
2. **Floating Action Button**: Quick access to add
3. **Modal Dialogs**: Focus on task at hand
4. **Smooth Transitions**: Delightful interactions
5. **Color Coding**: Quick status recognition

### Data Choices
1. **Local-first**: Privacy by default
2. **Dexie**: Simple, powerful IndexedDB wrapper
3. **JSON Export**: Easy backup & migration
4. **Normalized**: Scalable schema
